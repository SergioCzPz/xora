This implementation uses [Angular](https://angular.dev) and [Tailwind](https://tailwindcss.com).

---

## Installation

1. You can clone this repository

```bash
git clone https://github.com/SergioCzPz/xora.git
```

2. Install all dependencies

```bash
npm install
```

3. Start dev mode

```bash
ng serve -o
```
